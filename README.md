# 国民保護関連情報 即時伝達システム（GAS）

本リポジトリは、地震・津波等の国民保護関連情報を取得し、
条件に応じて即時通知を行う Google Apps Script（GAS）サンプル実装です。

本ソフトウェアは **有償提供** を前提としており、
オープンソースソフトウェアではありません。

---

## 想定用途
- 自治体・自主防災組織における情報伝達訓練
- 国民保護関連情報の技術検証
- 防災情報配信システムの研究・学習用途

※ 実運用・住民向け配信を保証するものではありません。

---

## 主な機能
- 地震情報（最大震度）自動取得
- 津波警報・注意報の即時通知
- 特定市町村の震度監視
- 条件判定によるメール通知
